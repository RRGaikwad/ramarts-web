RamArts — How to add your own images
=====================================

Put your image files HERE (this folder), for example:
  assets/images/shop-board.jpg
  assets/images/led-sign.png

In the Admin panel, use this path:
  assets/images/shop-board.jpg

DO NOT use Windows paths like:
  C:\Users\HP\Pictures\photo.jpg   (will NOT work in browser)

Other options in Admin:
  1. Click "Upload" — saves image in browser (works immediately)
  2. Paste a full web link: https://yoursite.com/image.jpg

Always run the site with a local server (npx serve .) when testing.
